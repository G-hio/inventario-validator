# inventario-validator
Sistema en Python para validar inconsistencias entre inventario físico y registros digitales en una base de datos MySQL.

## Requisitos
- Python 3.8+
- MySQL (server)
- pip install -r requirements.txt

## Configuración
1. Crear una base de datos MySQL y ejecutar `create_db.sql`.
2. Editar `config.ini` con los datos de conexión.
3. Ejecutar `python main.py` para validar inventario.
